# 🌊 Water Ripple

An interactive HTML5 Canvas animation that creates realistic water ripple effects with smooth expanding waves and fluid motion. Designed for both desktop and mobile devices, users can generate beautiful ripple patterns through mouse clicks, touch interactions, and continuous drag movements.

## ✨ Features

* 🌊 Realistic water ripple effects
* 💧 Smooth expanding wave animations
* ✨ Dynamic fading ripple trails
* 🖱️ Mouse click and drag support
* 📱 Mobile touch interaction
* 🎨 Fluid visual effects
* ⚡ Real-time Canvas rendering
* 🖥️ Fullscreen responsive design
* 🚀 Lightweight and dependency-free

## 🎮 Controls

### Desktop

* Click anywhere to create water ripples.
* Hold and drag to continuously generate ripple waves.

### Mobile

* Tap anywhere to create ripple effects.
* Touch and drag to generate continuous water wave animations.
